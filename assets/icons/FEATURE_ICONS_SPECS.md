# Feature Icons Specifications — Don't Click That

## Overview
8 custom icons representing core features of Don't Click That. Designed for consistency, clarity, and senior-friendly recognition.

---

## Design System

### Global Specifications
- **Style:** Line art with optional fill (dual-tone)
- **Stroke weight:** 3px (thick enough for visibility at small sizes)
- **Corner radius:** 4px (rounded, friendly)
- **Grid:** 128×128px canvas with 16px padding (96×96px active area)
- **Color palette:** Trust Blue (#2563EB), Safety Green (#10B981), Alert Red (#EF4444)
- **Format:** SVG (primary), PNG (128px, 256px, 512px)

### Visual Language
- **Simple shapes:** Avoid complex details
- **High contrast:** Works on light and dark backgrounds
- **Metaphorical:** Immediately recognizable symbols
- **Consistent:** Uniform stroke weight and style across all icons

## Icon Specifications

### 1. Email Protection
**Metaphor:** Secure email delivery

**Visual Description:**
- **Primary element:** Envelope (front view)
  - Outline: Trust Blue (#2563EB), 3px stroke
  - Flap: Indicated by diagonal line
  - Size: 70×50px centered
- **Secondary element:** Shield badge (overlay, bottom-right)
  - Size: 35×40px
  - Fill: Safety Green (#10B981)
  - Icon: White checkmark (✓) inside shield
  - Slight overlap with envelope (10px)
- **Additional:** Subtle lock icon on envelope flap (optional)

**Color Variants:**
- **Primary:** Blue envelope + green shield
- **Monochrome:** Single blue, different opacities
- **Dark mode:** Lighter blue (#3B82F6) + bright green (#34D399)

**Message:** "Your email is protected and verified safe"

### 2. Call Screening

## Usage Guidelines

### Sizing
- **Extra Small (32px):** Icon only, high contrast
- **Small (64px):** Icon only, standard use
- **Medium (128px):** Icon + optional label below
- **Large (256px+):** Icon + label + description

### Spacing
- **Icon grid:** 128×128px total, 16px padding = 96×96px active area
- **Label spacing:** 12px gap between icon and label text
- **Group spacing:** 24px minimum between multiple icons

### Color Variations

**Primary (Light Background):**
- Main: Trust Blue (#2563EB)
- Accent: Safety Green (#10B981)
- Alert: Alert Red (#EF4444)
- Background: Transparent or white

**Dark Background:**
- Main: Lighter Blue (#3B82F6)
- Accent: Bright Green (#34D399)
- Alert: Bright Red (#F87171)
- Background: Transparent or dark gray

**Monochrome:**
- Single color (Trust Blue or black)
- Varying opacity for depth (100%, 70%, 40%)

### Accessibility
- **Contrast ratio:** Minimum 4.5:1 against background
- **Alternative text:** Each icon must have descriptive alt text
- **Don't rely on color alone:** Use shapes and symbols

## Implementation

### Web Usage
```html
<div class="feature-icon">
  <img src="dct-icon-email-protection-128.svg" 
       alt="Email Protection - Shield with checkmark over envelope" 
       width="128" height="128">
  <h3>Email Protection</h3>
</div>
```

### CSS Styling
```css
.feature-icon {
  width: 128px;
  height: 128px;
  display: inline-block;
}

.feature-icon img {
  filter: drop-shadow(0 4px 6px rgba(0, 0, 0, 0.1));

  transform: scale(1.05);
  transition: transform 0.2s ease;
}
```


## Animation Guidelines (Optional)

### Hover States
- **Scale up:** 105-110% on hover
- **Color shift:** Brighten colors by 10%
- **Glow effect:** Subtle outer glow (8px, 20% opacity)
- **Duration:** 0.2s ease transition

### Loading States
- **Pulse:** Opacity 100% to 70% and back (1.5s loop)
- **Spin (for real-time icon):** 360° rotation (2s loop)
- **Checkmark draw:** Animate checkmark stroke (0.5s)

### Success States (when feature active)
- **Green fill:** Icon fills with Safety Green
- **Checkmark overlay:** Brief checkmark animation
- **Haptic feedback:** On mobile devices
