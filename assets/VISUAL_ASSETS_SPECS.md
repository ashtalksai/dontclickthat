# Don't Click That — Visual Assets Specifications
**Project:** Don't Click That — AI Scam Protection for Seniors  
**Date:** March 2, 2026  
**Stage:** 9 - Create Visuals  

## Brand Identity

### Color Palette
- **Trust Blue:** #2563EB (Primary brand color)
- **Safety Green:** #10B981 (Success, protection indicators)
- **Alert Red:** #EF4444 (Warning, threats)
- **Warning Orange:** #F97316 (Caution)
- **Neutral Gray:** #6B7280 (Text, borders)
- **Light Background:** #F9FAFB
- **Dark Background:** #111827

### Typography
- **Headings:** Clear, bold, highly legible (Inter, SF Pro Display)
- **Body:** Large text (minimum 16px) for senior-friendly readability
- **Style:** Professional, trustworthy, approachable (not corporate-cold)

### Design Philosophy
- **Senior-Friendly:** Large, clear elements with high contrast
- **Trustworthy:** Professional without being intimidating
- **Protective:** Shield imagery, safety-first visual language
- **Simple:** Clean layouts, minimal clutter

## 1. Logo Design

### Primary Logo Concept
**Visual Description:**
- **Icon:** Shield shape with a cursor/pointer icon crossed out with an X
- **Shield:** Rounded corners, Trust Blue (#2563EB) fill
- **Cursor:** White/light gray cursor pointer with red X overlay
- **Badge style:** Modern, geometric, clean lines
- **Size:** Works from 32px to 512px without loss of clarity

**Lockup:**
```
[Shield+X-Cursor Icon] Don't Click That
                       Scam Protection
```

### Logo Variations

#### A. Full Lockup (Horizontal)
- Icon + "Don't Click That" wordmark + tagline
- Use: Website header, email signatures, presentations
- Dimensions: 400x120px (scalable)

#### B. Icon + Wordmark Only
- Icon + "Don't Click That" text (no tagline)
- Use: App icon, social media profile, compact spaces
- Dimensions: 300x100px (scalable)

#### C. Icon Only (Badge)
- Just the shield with crossed-out cursor
- Use: Favicon, app icon, watermarks
- Dimensions: 512x512px square

#### D. Light Background Version
- Trust Blue shield (#2563EB)
- White cursor with red X
- Dark text (#111827)

#### E. Dark Background Version
- Lighter blue shield (#3B82F6)
- White cursor with bright red X (#EF4444)
- White text

#### F. Monochrome Version
- Single color (Trust Blue or black)
- For print, fax, black & white contexts

### Technical Specifications
- **Format:** SVG (primary), PNG (512px, 1024px, 2048px)
- **Safe zone:** 10% padding on all sides
- **Minimum size:** 32px height (icon only), 80px height (with text)

**Style:** Line-art with fill, 3px stroke weight, rounded corners

**Icon Descriptions:**

1. **Email Protection**

   - Shield icon blocking red "X" threat

3. **Family Dashboard**
   - Multiple user circles connected to central monitor/screen
   - Dashboard interface with metrics
   - Color: Trust Blue, multi-colored user avatars

4. **Privacy-First**
   - Lock icon with keyhole
   - Eye with "privacy" slash overlay
   - Color: Trust Blue lock, subtle gradient

5. **Real-Time Protection**
   - Clock/stopwatch with shield
   - Speed lines indicating immediate action
   - Color: Trust Blue, Safety Green accents

6. **Plain English Explanations**
   - Speech bubble with "ABC" or simple text
   - No complex symbols inside
   - Color: Trust Blue outline, white fill

7. **99.7% Accuracy**
   - Target/bullseye with checkmark
   - Percentage indicator
   - Color: Trust Blue target, Safety Green checkmark

8. **Senior-Friendly Design**
   - Large button or interface element
   - Hand/finger pointing or tapping
   - Color: Trust Blue UI, warm skin tone for hand

## Brand Guidelines Summary

### Do's
✅ Use high contrast for senior-friendly readability  
✅ Keep layouts clean and uncluttered  
✅ Use shield/protection imagery consistently  
✅ Emphasize trust and safety  
✅ Show real benefits (not just features)  

### Don'ts
❌ Don't use small text (minimum 16px)  
❌ Don't use low-contrast colors  
❌ Don't show scary/fear-inducing imagery  
❌ Don't use complex jargon or technical terms  
❌ Don't clutter designs with too many elements
