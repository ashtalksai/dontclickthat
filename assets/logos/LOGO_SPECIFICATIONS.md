# Logo Specifications — Don't Click That

## Primary Logo Concept

### Visual Design
The logo combines a **shield icon** (protection, trust) with a **crossed-out cursor** (don't click) to create an immediately recognizable symbol of digital protection.

### Detailed Description

**Shield Element:**
- Shape: Rounded shield (classic heraldic shape with modern rounded corners)
- Dimensions: 100px width × 120px height (at base size)
- Fill: Trust Blue (#2563EB) with subtle gradient to lighter blue (#3B82F6)
- Border: 2px stroke, slightly darker blue (#1E40AF) for definition
- Style: Flat design with subtle depth (1-2px shadow for dimensionality)

**Cursor Element:**
- Classic mouse pointer/arrow shape (white fill)
- Size: 40px height, positioned center-right of shield
- Angle: Standard 45° northwest pointing direction

**X-Mark (Prohibition Symbol):**
- Bold red X (#EF4444) overlaying the cursor
- 6px stroke width for visibility
- Diagonal cross (45° angles)
- Slightly extends beyond cursor bounds for emphasis

**Visual Hierarchy:**
1. Shield (background layer) — establishes trust
2. Cursor (middle layer) — represents clicking/interaction
3. Red X (foreground layer) — clear prohibition message

### Color Specifications

**Primary (Light Background):**
- Shield: #2563EB (Trust Blue)
- Gradient highlight: #3B82F6 (lighter blue, 20% opacity)
- Cursor: #FFFFFF (white) with #E5E7EB (light gray) shadow
- X-mark: #EF4444 (Alert Red)
- Border: #1E40AF (darker blue)

**Dark Background Variant:**
- Shield: #3B82F6 (brighter blue for contrast)
- Gradient: #60A5FA (even lighter)
- Cursor: #FFFFFF (white) with glow effect
- X-mark: #F87171 (brighter red)
- Border: #2563EB

**Monochrome:**
- Single color: #2563EB or #000000
- All elements same color with varying opacity for depth

## Typography Lockup

### Wordmark: "Don't Click That"
**Font:** Inter Bold or SF Pro Display Bold
- Weight: 700 (Bold)
- Size: 48px (at base 300px width)
- Color: #111827 (dark gray, almost black)
- Letter spacing: -0.02em (tight, modern)
- Case: Title Case

### Tagline: "Scam Protection"
**Font:** Inter Medium or SF Pro Display Medium
- Weight: 500 (Medium)
- Size: 18px
- Color: #6B7280 (medium gray)
- Letter spacing: 0.05em (slightly wider for readability)
- Case: Title Case
- Position: Below wordmark, left-aligned with icon

## Technical Requirements

### File Formats
1. **SVG (Master):**
   - Vector format, infinitely scalable
   - Text as outlines (for font compatibility)
   - Organized layers: Background, Icon, Text

2. **PNG (Export):**
   - 512px (standard web)
   - 1024px (retina/high-DPI)
   - 2048px (print quality)
   - Transparent background

3. **Favicon Package:**
   - 16×16px, 32×32px, 64×64px
   - ICO format for browser compatibility
   - Apple Touch Icon: 180×180px

### Safe Zones
- **Minimum clearance:** 10% of logo height on all sides
- **No elements** (text, graphics, other logos) within safe zone
- **Background:** Ensure sufficient contrast (WCAG AA minimum)

### Minimum Sizes
- **Icon only:** 32px height (favicon, small UI)
- **Icon + wordmark:** 80px height (header, navigation)
- **Full lockup:** 100px height (hero, primary placements)

## Usage Guidelines

### ✅ Correct Usage
- Maintain aspect ratio (never stretch or distort)
- Use approved color variations only
- Respect safe zones and minimum sizes
- Ensure sufficient background contrast
- Use provided file formats (don't recreate)

### ❌ Incorrect Usage
- Don't change colors (unless using approved variants)
- Don't rearrange elements
- Don't add effects (drop shadows, gradients, outlines)
- Don't rotate or skew
- Don't place on busy backgrounds without container
- Don't use low-resolution versions for print

## Background Recommendations

### Light Backgrounds
- White (#FFFFFF)
- Light gray (#F9FAFB, #F3F4F6)
- Light blue (#EFF6FF)
- Use primary logo variation

### Dark Backgrounds
- Dark gray (#111827, #1F2937)
- Dark blue (#1E3A8A)
- Black (#000000)
- Use dark background variant (brighter colors)

### Colored Backgrounds
- Avoid busy patterns
- If necessary, place logo in white/dark container box
- Maintain WCAG AA contrast ratios

## Accessibility

### Color Contrast
- Shield vs. background: Minimum 4.5:1 ratio
- Text vs. background: Minimum 4.5:1 ratio (AA standard)
- X-mark vs. cursor: High contrast for visibility

### Alternative Text
- "Don't Click That logo — AI scam protection for seniors"
- Use in HTML alt attributes and accessibility labels
